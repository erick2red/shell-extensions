### Gnome-Shell Extensions

#### For now
* workspace-indicator extension
* wish-list extension

#### Requirements
* Gnome-Shell 3.0.1 or later

#### Hwo to install
* Open a console and type the following:
	* git clone git://github.com/erick2red/shell-extensions.git
	* cd shell-extensions
	* ./autogen.sh
	* make
	* sudo make install
* Then restart the Shell
